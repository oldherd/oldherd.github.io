花了那么长时间再加上多次死皮赖脸地找出题人提问终于刚掉了这题。出题人的题解很棒，但是刚接触的期望萌新并不是很够智商。不过在出题人的耐心帮助下还是解决了。
解决本题的关键在于对整体的状态空间的把握。对于一种元素,有转化成最终元素(以下或简记为成功转化)的情形和不成功转化的情形。而对于本数不同的属性成功的概率是不同的,也就意味着在状态空间内,每种属性成功的情况的数量不相同。
对于任意钦定的魔法属性,若在状态中的个数为iii,其为最终结果的概率设为pip_ip​i​​。因为在最终这个状态还是会转化出去的(除非i=ni = ni=n),而向两边转化的概率是相等的,所以 pi=pi−1+pi+12p_i = \frac{p_{i-1} + p_{i+1}}{2}p​i​​=​2​​p​i−1​​+p​i+1​​​​。又由pn=1p_n = 1p​n​​=1, p0p_0p​0​​ = 0。故而pi=inp_i = \frac{i}{n}p​i​​=​n​​i​​。
设在取得成功转化的条件下步数的期望为fif_if​i​​，iii代表该属性下书的数量。我们考虑将它转移。转移的式子是fi=n(n−1)2i(n−i)+i−12ifi−1+i+12ifi+1f_i = \frac{n(n-1)}{2i(n-i)} + \frac{i-1}{2i}f_{i-1} + \frac{i+1}{2i}f_{i+1}f​i​​=​2i(n−i)​​n(n−1)​​+​2i​​i−1​​f​i−1​​+​2i​​i+1​​f​i+1​​。下面来解释这个式子。
由古典概型,转移出去的概率是2i(n−i)n(n−1)\frac{2i(n-i)}{n(n-1)}​n(n−1)​​2i(n−i)​​,所以转移出去的期望步数为它的倒数n(n−1)2i(n−i)\frac{n(n-1)}{2i(n-i)}​2i(n−i)​​n(n−1)​​,这是转移的代价。接下来,要注意到我们的fif_if​i​​定义为成功条件下的期望步数,所以我们的转移也是在成功的状态空间内进行的。这样,根据我们在开头提到的,转移成功概率不相等意味着在成功条件下的状态空间内的数目不相等,所以在成功条件下的状态空间中转移到fi−1f_{i-1}f​i−1​​和fi+1f_{i+1}f​i+1​​的概率也是不相等的。它们在成功的状态空间内的数量比应该等于其成功转化的概率比。而我们在成功条件下转化到i−1i-1i−1和i+1i+1i+1的概率也就是i−1i-1i−1和i+1i+1i+1在成功条件下在状态空间中的数目占比。
