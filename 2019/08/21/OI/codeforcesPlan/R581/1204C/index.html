Codeforces Round581 1204C题解

OI-题解
  |  
 阅读次数

题意简述:以邻接矩阵的形式给定一张有向无环图无权图,1表示有边0表示没有。然后给定一条路径(不一定为简单路径),按照遍历的点的顺序给出。要求:尝试在路径中删除尽可能多的点,使得要按顺序走完剩余的结点(可以经过其他结点)所走的最短路径长度等于原路径长度。
来看数据:2<=n<=1002<= n <=1002<=n<=100,暗示O(n3n^3n​3​​)，m<=106m <= 10^6m<=10​6​​暗示O(mlogmmlogmmlogm)或者O(m)
O(n3n^3n​3​​)可以跑个floyd。O(mlogmmlogmmlogm)或者O(m)只能一遍扫下来在扫的过程中决策。
一个点能够被删除的条件是什么？如果前面一个保留的点到这个点后一个点的最短路径等于前面一个点经过这个点到这个点后面一个点的最短路径相等,那么这个点可以被删除。换言之,在路径p1p_1p​1​​ p2p_2p​2​​ … pkp_kp​k​​ … pip_ip​i​​ pi+1p_{i+1}p​i+1​​ …pmp_mp​m​​ 我们的决策已经进行到pip_ip​i​​,上一个没有被删除的点是pkp_kp​k​​，那么pip_ip​i​​能被删除的条件是:从pkp_kp​k​​按原路径走到pi+1p_{i+1}p​i+1​​的长度等于从pkp_kp​k​​走到pi+1p_{i+1}p​i+1​​的最短路径长度。其中,pkp_kp​k​​按原路径走到pi+1p_{i+1}p​i+1​​的路径长度等于i−ki-ki−k，而pkp_kp​k​​到pi+1p_{i+1}p​i+1​​的最短路径长度我们可以采用floyd O(n3n^3n​3​​)预处理出来。而如果后者小于前者,则pip_ip​i​​不能被删除。
