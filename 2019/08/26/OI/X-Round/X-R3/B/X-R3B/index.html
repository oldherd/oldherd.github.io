XR-3 T2 小道消息

OI-题解
  |  
 阅读次数

比赛时打错素数判定，查错无果后自闭弃赛回房间看番
结论的话,传到话的人号码是素数,并且这个数字的两倍大于n,答案为1,否则为2.
伪证(以下涉及的序号均为衣服上的数字而非人的序号):
设号码为ttt, t=k+1t = k + 1t=k+1.若t为素数,则它与2...2t2...2t2...2t-111的除了ttt外的数均互质,故而答案为1。
然后,若t为素数但是2t<=n2t <= n2t<=n,这样,1次传话后未被传达的数为nt,t∈{2,3,4,...}nt, t\in \{2, 3, 4, ...\}nt,t∈{2,3,4,...}。由于t为素数,故t>1t > 1t>1.所以这些数会在第2天传话中被t−1t-1t−1或t+1t+1t+1传到。
若ttt不为素数,第一轮传话之后,没有被传到话的人包括:ttt的所有质因数,ttt的质因数的正整数倍。但是,t−1t-1t−1与这些数字都是互素的,所以这些数字都会在第2天被传到。
这些东西是看完番后11点钟证的。考试的时候,其实1和2的答案只是代了几组数据瞎猜的。然后打错入门难度的质数判断以为自己想锅了所以在8点就弃赛了QaQ.
