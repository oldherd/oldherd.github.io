小a和uim之大逃离 LuoguP1373

OI
  |  
 阅读次数

观察

方案数辣么多,正解应当在 搜索-dp 或者 数学/数论 中找。
于是根据直觉 搜索-dp。

状态设计

$f_i,j,t,0|1$代表在坐标i,j使两人取得的魔力在模k+1意义下差值(小a-小uim)为t,当前格由0:小a,1:小uim收取在模1e9+7意义下的方案数。

转移

$f_i,j,t,0 = f_i-1,j,(t-a[i][j]+k+1)%(k+1),1 + f_i,j-1,(t-a[i][j]+k+1)%(k+1),1 t \not a[i][j]$
$f_i,j,t,0 = f_i-1,j,(t-a[i][j]+k+1)%(k+1),1 + f_i,j-1,(t-a[i][j]+k+1)%(k+1),1 + 1, t == a[i][j]$
$f_i,j,t,1 = f_i-1,j,(t+a[i][j])%(k+1),0 + f_i,j-1,(t-a[i][j]+k+1)%(k+1),0$

边界

f_i,j,a[i][j],0$为出发点设为1(已经体现在了状态转移方程上面)。
