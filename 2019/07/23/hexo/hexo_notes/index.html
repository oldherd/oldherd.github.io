本文主要记录日常用到过的hexo方法。

1.文件开头

2.yml

hexo的基本配置信息在_config.yml中更改。
如:

3.theme

主题的下载大致采用如下格式:
git clone + 网址 + themes/+主题名称
主题中可以更改一些东西，如背景等，应该在themes下的主题文件夹中的source/halmenu中更改
打开主题的excerpt(部分主题已经默认打开)，在文章中某部分加入<!-- more -->可以使在主页只加载这之前的部分作为预览。

4.新建页面

使用命令hexo new page "xxxxxx"新建对象名为xxxxxx的页面。会在sourse下创建一个同名新文件夹,内有文件index.md,在此处编辑新页面的内容。
在应用的主题的menu栏仿照按照格式填写xxx: /xxxxxx。就会在Sider Bar上添加名为xxx的导航链接。xxx可以为中文。

5.关于图片

官方的解决方式是添加插件。这里的解决方式是在source文件夹下创建子文件夹images然后在需要处使用绝对地址引用(除了在全局的css中可以使用相对地址,如滚动背景的css)。文件夹images位于网页的根目录中。

6.发布

hexo clean && hexo g && hexo d
