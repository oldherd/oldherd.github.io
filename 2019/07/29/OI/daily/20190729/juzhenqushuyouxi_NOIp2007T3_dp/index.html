矩阵取数游戏 NOIp2007_T3 LuoguP1005 题解

OI
  |  
 阅读次数

每一行都是一个区间dp。这点不难看出来。

状态设计

直接对取数设计dp有点困难。
hzwer: 正难则反。
不妨将取数字看作放数字。放第一个数的权值为$2^m$,第二个数权值为$2^m-1$,依此类推。
那么放的规则为:除了第一个数字外,新放入的数字应当放在原有数字串的两端。
记$f_l,r$为放入区间[l, r]的最大收益。

转移方程:

$f_l,r = max{ f_l+1,r + a_i * 2^m-r+l, f_l,r-1 + a_j * 2^m-r+l }$。

边界条件:

区间dp的边界一般为$len = 1$(len代表区间长度即r-l+1)。
$f_l,l = a_l * 2^m$

数组使用

不要开三维数组,会爆掉空间。考虑到每一行的独立性,重复利用f数组,只要每一次把$len = 1$行的初值赋好即可。

预处理

可以考虑打出2的幂次方的表格,最大到$2^m$即可。在O(m)的时间递推即得。

关于范围

算一下,答案最大可到40多位数(或许不到一点)。因而打一个高精度。
max函数可以直接采用algorithm库里的,只要重载好大于号即可。
