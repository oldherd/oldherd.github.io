SuperGCD SDOI2009 LuoguP2152 题解(含有关于更相减损术的步骤描述)

OI
  |  
 阅读次数

hzwer: 更相减损术在时间上比较慢,一般用于证明一些结论和在高精度下求GCD。
本题数据范围在高精度的范畴,由于高精度除法(取模)难度较大,但是高精度相减而高精度除以单精度、乘以单精度在实现上相对简单,所以考虑用更相减损术来实现。
更相减损术的实现:
1.若两个整数都为偶数,将他们不断同时除以2，直到其中任意一个不为偶数。记录除以2的次数。
2.若其中有任意数为偶数,将其除以2,直到没有偶数。
3.用大的数减去小的数,以之代替较大的数。
4.重复2、3直到两个数字不相同为止。
5.将剩下的数字乘以在步骤1中除掉的2,即得gcd。
