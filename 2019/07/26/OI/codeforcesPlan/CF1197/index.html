Educational codeforces Round 69(Rated for Div. 2), contest 1197 题解

OI
  |  
 阅读次数

话说上一回的Regular Round 574题解还没更完，但是又开了一本(画风有点不对?)
因为从不修仙(有人信吗?)刷题所以没办法参加比赛只能赛后打打。所以您看到的一直是Div. 2(说的好像Div. 1我能做出几道一样)。
这次比赛几乎没有考察高级的算法，全都是智商题(其实就是水)。
这次首更四题

CF1197A DIY Wooden Ladder

我们要做的就是构造出解。
k最大能到多少呢?至多到比第二小的数小一。否则不满足题意。这是必要条件。
然后我们要选出k+2个数,意味着k <= n-2,否则数字根本不够选。
取二者最小值即为答案。

CF1197B Pillars

一个盘子只能被移动一次,因为盘子被移动后移动到的柱子就有超过一个的盘子了,那么这根柱子上的东西显然不能再移动了。
所以只有当盘子在一个方向上单调增大,这列盘子才可以叠在一起。
具体到题目中,全部的盘子只有是单调递增,或单调递减,或单峰(先单调递增后单调递减),才能被叠成一堆。
我们的程序只要检验单调性即可。

CF1197C Array Splitting

这道题目乍一看是一个区间dp。
设dp(i, l, k)为区间[i, l+1-i]划分为k段(l为区间长度)。递推式为:dp(i, l, k) = min{dp(i, t1, t2) + dp(i+t1, l-t1, k-t2), dp(i, t1, t2) + dp(i+t1, l-t1, k-t2+1)} (1 <= t1 <= l, 1 <= k2 <= t)。
显然,状态数为O($n^2 * k$), 转移为O(n * k), 时间复杂度为O($n^3 * k^2$), 空间复杂度为O($n^2 * k$)。横着看竖着看怎么都过不了。
本来是考虑研究性质优化dp的，没想到想出了正解。
一个单调的数列$a_1 … a_n$,看作一段,极差就是$a_n - a_1$, 记为S。如果在$a_l$后面分段,和就变成了$S-(a_l+1 - a_l)$。由此可见,我们要使k段极差和最小,就要在相邻差值最大的k-1对数字之间划分。这题考察贪心。

CF1197D Yet Another Subarray Problem

首先考虑$m = 1$的情况。我们发现$ \sum^{i = l}_{r}{a_i} - k \left\lceil\frac{r-l+1}{m}\right\rceil = \sum{i=l}^r (a_i-k)$,相当于在原数组减k求最大字段和。
$m \not 1$时,原式 = $a_1 - k + a_2 + … + a_m + a_m+1 - k + a_m+2$。那么是对下标模m余1的数求最大字段和吗?
我们发现,当某处开始的前缀和为非正数时,如果一般的最大字段和重新开始计算就好了。但是这里有个问题,如果重新计算,那么减k的位置也要变动。也就是说,可能存在类似这样的情况,$a_l+ … + a_km+1 - k < 0, a_km+2 -k < 0$, 但是$a_l + … a_km+1 - k + a_km+2 > 0$。也就是说,我们在某处为起点的前缀和统计值为负数时,有两种最优解的走向。一种是舍弃之前的部分,另一种是接着继续求下去。这样子,我们有很多个分叉,分叉之后还有分叉。很难实现。
遇到这类问题一般考虑用dp试试看,不行就搜索。
我们设$f_i,j$为以i为结尾,下标模m为j的最大字段和。(注意,为了表示方便,令选中子段的第一个数下标为0,也就是说j=0时减k)。那么当$j \not 0$时, $f_i,j = f_i-1,j-1$。当$j = 0$时, $f_i,0 = max{f_i-1,m-1 , a_i} - k$(分别代表重开和继承两种选项,这就是dp的优点,它可以帮我们融合复杂的选项,根据无后效性(指当前的选择仅影响当前的最优解选择,并不影响后面得出最优解)原理。)
关于边界条件,首先$f_0,0 = a_0$, 有一些值不可能取到,比如f_0,1,这显然是不可能的。我们把除了f_0,0的地方全部赋初值为负无穷,要算的地方就不会由他们更新得到,只会由f_0,0与合法更新得到的值更新得到。
