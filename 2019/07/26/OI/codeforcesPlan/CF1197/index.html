Educational codeforces Round 69(Rated for Div. 2), contest 1197 题解

OI-题解
  |  
 阅读次数

话说上一回的Regular Round 574题解还没更完，但是又开了一本(画风有点不对?)
因为从不修仙(有人信吗?)刷题所以没办法参加比赛只能赛后打打。
这次比赛几乎没有考察高级的算法，全都是智商题(其实就是水)。
这次首更四题

CF1197A DIY Wooden Ladder

我们要做的就是构造出解。
k最大能到多少呢?至多到比第二小的数小一。否则不满足题意。这是必要条件。
然后我们要选出k+2个数,意味着k <= n-2,否则数字根本不够选。
取二者最小值即为答案。

CF1197B Pillars

一个盘子只能被移动一次,因为盘子被移动后移动到的柱子就有超过一个的盘子了,那么这根柱子上的东西显然不能再移动了。
所以只有当盘子在一个方向上单调增大,这列盘子才可以叠在一起。
具体到题目中,全部的盘子只有是单调递增,或单调递减,或单峰(先单调递增后单调递减),才能被叠成一堆。
我们的程序只要检验单调性即可。

CF1197C Array Splitting

这道题目乍一看是一个区间dp。
设dp(i, l, k)为区间[i, l+1-i]划分为k段(l为区间长度)。递推式为:dp(i, l, k) = min{dp(i, t1, t2) + dp(i+t1, l-t1, k-t2), dp(i, t1, t2) + dp(i+t1, l-t1, k-t2+1)} (1 <= t1 <= l, 1 <= k2 <= t)。
显然,状态数为O(n2∗kn^2 * kn​2​​∗k), 转移为O(n * k), 时间复杂度为O(n3∗k2n^3 * k^2n​3​​∗k​2​​), 空间复杂度为O(n2∗kn^2 * kn​2​​∗k)。横着看竖着看怎么都过不了。
本来是考虑研究性质优化dp的，没想到想出了正解。
一个单调的数列a1...ana_1 ... a_na​1​​...a​n​​,看作一段,极差就是an−a1a_n - a_1a​n​​−a​1​​, 记为S。如果在ala_la​l​​后面分段,和就变成了S−(al+1−al)S-(a_{l+1} - a_l)S−(a​l+1​​−a​l​​)。由此可见,我们要使k段极差和最小,就要在相邻差值最大的k-1对数字之间划分。这题考察贪心。
