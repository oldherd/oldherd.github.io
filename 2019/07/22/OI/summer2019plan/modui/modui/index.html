莫队算法练习

OI
  |  
 阅读次数

莫队算法将询问离线化，按区间右端点分块，然后暴力移动指针修改变化的位置的点。
莫队算法适用于能够O(1)维护删除一个点和增加一个点，统计类(统计量能够装入桶中)的题目，复杂度为O(n√n + m√n)。
一般块的大小为√n或者n / (√m * 2/3)，后者优于前者。
初始化将l = 1, r = 0.这样在它们移动到排序后的询问序列过程中的中间段分别被添加和删除各一次，正好抵消。

例题一:小B的询问 洛谷P2709 BZOJ3781

这是一道板子题。

例题二:小Z的袜子 洛谷P1949 BZOJ2038

由排列组合知识，答案为:(ΣC(n[i], 2)) / C(len, 2). 其中n[i]表示区间内i颜色的袜子出现的次数,len表示区间长度。 化简该公式可得:Σ(n[i] * n[i-1]) / (len*(len-1))。
前面的和式可以由莫队计算。约简则是求gcd然后两边同时除掉。
